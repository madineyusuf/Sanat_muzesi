##  Özellikler

- Kullanıcı kaydı ve girişi
- Sanat eserlerini listeleme ve arama
- Kategoriye göre filtreleme
- Eserleri favorilere ekleme / çıkarma
- Eserlere yorum yapma ve silme
- Profil bilgilerini güncelleme (kullanıcı adı, şifre)

##  Kullanılan Teknolojiler

- PHP 
- MySQL 
- HTML5
- Bootstrap 5
- Bootstrap Icons
- JavaScript 

##  Veritabanı Tabloları

| Tablo | Açıklama |
|---|---|
| `kullanicilar` | Kullanıcı bilgileri |
| `eserler` | Sanat eserleri |
| `favoriler` | Kullanıcı favorileri |
| `yorumlar` | Eser yorumları |

##  Kurulum

1. Repoyu klonlayın
2. `includes/db.example.php` dosyasını `includes/db.php` olarak kopyalayın
3. Veritabanı bilgilerinizi girin
4. `schema.sql` dosyasını veritabanınızda çalıştırın
5. Dosyaları sunucuya yükleyin
